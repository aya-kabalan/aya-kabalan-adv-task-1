# Aya's Digital Library
Welcome to my digital sanctuary for managing and cataloging books. This project is a front-end task focused on applying Object-Oriented Programming (OOP) concepts in web development.
#Features:
- Dynamic Cataloging:** Add and delete books on the fly.
- Search & Filter:** Search by title or author, and filter by categories.
- Full Responsiveness:** Optimized for desktop, tablets, and mobile phones.
- OOP Core:** Leveraged classes, encapsulation with private properties (`#`), and inheritance.
 #Technologies Used:
- **HTML5:** For the structured layout.
- **CSS3:** Custom styling with modern dark theme and CSS Grid.
- **TypeScript / JavaScript:** To handle logic, classes, and interactions
- ## How to Run the Project:
- Clone this repository to your local machine:
   ```bash
   git clone https://github.com/aya-kabalan/aya-kabalan-adv-task-1.git
   ```
   Open the index.html file in any modern web browser
